## Problem Statement
NexGen Logistics faces frequent delivery delays that impact customer satisfaction
and increase operational costs. Currently, delays are identified only after they occur.
This project focuses on predicting delivery delay risks before dispatch using
historical logistics data.

## Solution Overview
This Streamlit-based application analyzes order, delivery, route, and cost data to:
- Identify key delay drivers
- Visualize operational inefficiencies
- Predict delay risk for new orders
- Provide actionable business recommendations

## Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Data cleaning and multi-dataset merging
- Key performance indicators for delivery performance
- Four business-focused visualizations
- Interactive delay risk prediction model
- Exportable processed dataset

## How to Run
1. Install dependencies  
   `pip install -r requirements.txt`

2. Run the application  
   `streamlit run app.py`
